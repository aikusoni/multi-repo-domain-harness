# 프로젝트 식별 등록부

하네스에 참여하는 저장소와 횡단 역할을 등록한다. 여기에는 상세 시스템 구조가 아니라 프로젝트의
책임과 소스 오브 트루스 위치만 짧게 기록한다. 상세 관계는 `docs/project-map.md`에서 관리한다.

## 작성 형식

```markdown
### <project-id>
- **종류**: repository | role
- **책임**: 이 프로젝트가 소유하는 도메인 또는 기능
- **소스 오브 트루스**: 저장소 내부 문서나 경로
- **관련 도메인**: domain-a, domain-b
```

프로젝트 식별자는 디렉터리명, 요청의 `target`/`from`, 작업 큐 경로, 변경 피드의 `영향`에서 동일하게
사용한다. 영문·숫자·점·밑줄·하이픈만 사용하는 것을 권장한다.
